# 📩 Spam Message Classifier using Machine Learning

## 📌 Overview
This project is a simple machine learning application that classifies text messages as Spam or Not Spam (Ham).

It helps identify unwanted or harmful messages automatically.

---

## ❗ Problem Statement
Users receive spam messages like fake offers or scams. This project detects such messages using ML.

---

## 💡 Solution
The system:
- Learns from sample messages
- Converts text into numbers
- Predicts whether a message is Spam or Ham

---

## 🧠 Technologies Used
- Python
- Pandas
- Scikit-learn
